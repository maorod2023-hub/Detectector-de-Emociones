# Final Project Emotion Detection Web Application

Proyecto Final: Presentación y Evaluación
Para este proyecto final – Detector de Emociones, las tareas a continuación asumiendo el rol de un ingeniero de software que desarrolla una aplicación web basada en IA.

Tarea 1: Clonar el repositorio del proyecto
Tarea 2: Crear una aplicación de detección de emociones utilizando la biblioteca Watson NLP
Tarea 3: Formatear la salida de la aplicación
Tarea 4: Empaquetar la aplicación
Tarea 5: Ejecutar pruebas unitarias en tu aplicación
Tarea 6: Despliegue web de la aplicación utilizando Flask
Tarea 7: Incorporar manejo de errores
Tarea 8: Ejecutar análisis de código estático
